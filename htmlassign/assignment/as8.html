HTML Elements Showcase – Inline vs Block

In HTML, elements are broadly divided into two categories: block-level and inline. Block elements take up the full width of their container and start on a new line, while inline elements only take as much space as needed and do not break the flow of text.

[Block Elements] [Inline Elements] [Comparison]

Block Elements

Example 1: <div>

This is a div container.

The <div> is a block-level container used to group content.

Example 2: <p>

This is a paragraph of text.

The <p> defines a block of text and always starts on a new line.

Example 3: Headings

Heading H1

Heading H2

Heading H3

Heading H4

Heading H5
Heading H6

Headings (<h1>–<h6>) are block elements used for document structure.

Example 4: <ul> and <ol>

  • Unordered item 1
  • Unordered item 2
  1. Ordered item 1
  2. Ordered item 2

Lists are block-level structures for ordered and unordered content.

Example 5: <blockquote>

"Coding is the new literacy."

The <blockquote> is used for long quotations.

Example 6: <hr>

The <hr> tag inserts a horizontal line across the page.

Example 7: <table>

Row 1, Col 1Row 1, Col 2
Row 2, Col 1Row 2, Col 2

The <table> element defines a block-level table.

Inline Elements

Example 1: <span>

This is inside a span element.

The <span> is an inline container with no new line.

Example 2: <a>

Visit Google for searching.

The <a> defines hyperlinks inline.

Example 3: Text Formatting

Bold, Italic, Underline, Strong, Emphasis

These are inline formatting elements applied inside text.

Example 4: Superscript & Subscript

E = mc2 and H2O

The <sup> and <sub> place text above or below baseline.

Example 5: <mark>

This is a highlighted word.

The <mark> highlights text inline.

Example 6: <abbr>

HTML is a markup language.

The <abbr> tag defines abbreviations with tooltips.

Example 7: <code>

Example code: console.log("Hello World")

The <code> represents inline programming code.
